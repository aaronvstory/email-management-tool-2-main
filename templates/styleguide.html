3) Buttons

All buttons are 42px high. Sizes: default, .btn-sm (34px), .btn-lg (50px).

Use one primary action per area; use confirmToast for destructive actions.
<button class="btn btn-primary-modern">Save</button> <button class="btn btn-secondary">Cancel</button>

10) Do / Don't
  • Do keep buttons 42px tall and inputs styled with .input-modern.
  • Do use toasts for feedback and confirmToast for destructive actions.
  • Don't mix arbitrary inline colors; use the token set.
  • Don't use table-striped on dark backgrounds; wrap with .table-modern.
